# Recipilation

#### This is a simple single-page recipe blog application, made using the [MERN Stack](https://www.mongodb.com/mern-stack). 

#### The purpose of this project is to improve my web development skills using the MERN stack. However, in terms of UI/UX, the design might not be good (it should be responsive though). 

#### This project is my personal capstone project after learning web development from [The Complete 2023 Web Development Bootcamp by Angela Yu](https://www.udemy.com/course/the-complete-web-development-bootcamp/) course. For details about the exercises on that course, check out [this repository](https://github.com/Matthew1906/WebDevelopmentBootcamp)

#### I also used [Firebase Authentication](https://firebase.google.com/docs/auth/) to help me with user authentication, [ImageKit](https://imagekit.io/) to store images, [Tailwind CSS](https://tailwindcss.com/) for styling, [React Router](https://reactrouter.com/en/main) for client-side routing and [React Query](https://tanstack.com/query/v3/) to increase data fetching performance.
